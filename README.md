# 🚌 Bus Pass Portal

A modern Bus Pass Portal built to simplify the bus pass application and management process. The system allows users to apply for bus passes online, while administrators can verify applications and manage pass requests through a dedicated dashboard.

## ✨ Features
- User Registration & Login
- Online Bus Pass Application
- Admin Dashboard
- Application Verification
- Pass Approval & Management
- Responsive User Interface

## 🛠️ Technologies Used
- html
- CSS3
- JavaScript (ES6+)
- Git & GitHub

## 🎯 Purpose
This project demonstrates front-end development skills by building a responsive and user-friendly web application with a clean interface and efficient user workflow.
